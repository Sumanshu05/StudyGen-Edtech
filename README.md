# StudyGen

StudyGen is an ed-tech platform built with React and Tailwind CSS.

## Usage

To run the project locally:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the client:**
   ```bash
   npm start
   ```

3. **Start the server (separate terminal):**
   ```bash
   npm run server
   ```

4. **Start both together:**
   ```bash
   npm run dev
   ```

Open the project in your browser at [http://localhost:3000](http://localhost:3000) to view your project.

Create your React components and add your styles using Tailwind classes. You can also create new CSS files and import them into your components.

Update backend branding variables in `SERVER/.env`:
```env
APP_NAME=StudyGen
PUBLIC_APP_URL=http://localhost:3000
SUPPORT_EMAIL=
```
